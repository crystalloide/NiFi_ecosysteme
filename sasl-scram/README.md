# SASL/SCRAM-SHA-256

## Description

This is a deployment with no encryption but with SASL/SCRAM-SHA-256 authentication:

* 1 zookeeper
* 1 broker
* 1 connect
* 1 schema-registry
* 1 control-center

## How to run

Simply run:

```
$ just use <playground run> command and search for start.sh in this folder
```
